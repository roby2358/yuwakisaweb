# Game

Super is a game of a superhero in a perilous world. The gameplay should center around a player with some set of superpowers. See superpower.txt for a list of superpowers. It is ranked for frequency if you want to create a roller to randomly generate powers or sets of powers.

# Maps

The game is played on a series of maps that should be about 30x30. Ranged attacks are typically 5-8 and movement (except for super speed) is 4-6 or so

The game should be a series of events on maps like
- City block
- Bank interior
- Mad scientist's lab
- Alien spaceship
- grassy field
- Evil Lair
- Ritual Cult site

Enemies should range from groups of easy enemies with maybe 1-2 powers to low-grade superpowered criminals to epic supervillans

# Implementation notes

The game will be played in the brower as html, css, and js. See the included implementation for a hex and counter game
